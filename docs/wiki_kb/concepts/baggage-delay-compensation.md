# 行李延误赔付规则

## 赔付阶梯 (Tier Lookup)

| 档位 | 延误时长 | 赔付金额 |
|------|----------|----------|
| 第一档 | 6h ≤ T < 12h | 500 元 |
| 第二档 | 12h ≤ T < 18h | 1000 元 |
| 第三档 | T ≥ 18h | 1500 元 |

延误时长未达 6 小时 → **拒赔**。

## 延误时长计算
```
延误时长 = 行李实际签收时间 - 首次航班实际到达时间
```
- 航班到达时间以飞常准/航旅纵横/航空公司官方数据为准
- 多份签收证明以最晚时间为准

## 特殊规则
- **责任竞合**：同时申请行李延误+随身财产损失，仅按较高额度赔付
- **多件行李**：同一权益人多件行李同时延误，按一次事故核算
- **索赔校准**：索赔金额 > 核算金额 → 按核算金额；索赔金额 < 核算金额 → 按索赔金额

## 人工复核触发条件
1. 手写体/多语言材料无法识别
2. 行李归属争议
3. 签收时间矛盾
4. 结案后补件涉及大额补赔（>1000元）
5. 疑似材料伪造
